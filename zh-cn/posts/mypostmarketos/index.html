小米平板5Pro(XiaoMi Pad Elish)#

首先需要获取平板的root权限，建议用面具刷，然后解锁BL锁，最后在FastLoad模式下线刷。

PostmarketOS Linux的安装#

具体参考PostmarketOS Wiki

安装后的初步配置#

  1. 一个体验更好的shell，安装zsh

这是我的zsh配置文件：

# Alias
alias ra='yazi'
alias ne='fastfetch --cpu-temp'

# History
HISTFILE="$HOME/.zsh_history"
HISTSIZE=1024
SAVEHIST=1024
setopt HIST_IGNORE_DUPS
setopt HIST_IGNORE_SPACE
setopt EXTENDED_HISTORY

# Plugins
source /usr/share/zsh/plugins/zsh-autosuggestions/zsh-autosuggestions.plugin.zsh
source /usr/share/zsh/plugins/zsh-completions/zsh-completions.plugin.zsh

autoload -U compinit
compinit

eval "$(starship init zsh)"

装了几个插件，没有使用Oh-My—Zsh或者Zim等，个人感觉有点臃肿。

  1. 开箱即用的桌面环境（KDE）

KDE桌面环境非常省事，而且个人觉得比GNOME好用一点。

KDE的动态壁纸插件有点耗资源，且有点费电，另外，KDE支持autostart脚本，因此很容易将自己以前用的的DWM动态壁纸脚本移植到KDE上，该脚本内容如下，可以实现壁纸随时间变化的效果：

#!/usr/bin/env sh

sleep 2
TEMP=0
MINS=10
# Path of Wallpaper
location="/home/johnny/.config/wallpapers/"
wallpaper=`date +%H`.jpg

# Set Origin Wallpaper
plasma-apply-wallpaperimage $location$wallpaper

while true;do

    let TEMP++

    if [ $TEMP -eq $MINS ];
    then
        TEMP=0
        wallpaper=`date +%H`.jpg
        
        plasma-apply-wallpaperimage $location$wallpaper
    fi

    sleep 60
done &
  1. ssh连接github出现的端口问题(因为我使用的v2raya梯子开的tun模式)

~/.ssh/config

Host github.com
Hostname ssh.github.com
Port 443

然后如果出现ssh "permissions are too open"错误，一般是因为id_ras的权限太过开放，我们只需要将他改成只读模式chmod 400 ~/.ssh/id_rsa就行了。
